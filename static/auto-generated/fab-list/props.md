### activated

|                 |                                                                                                                                                    |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Description** | If `true`, the fab list will show all fab buttons in the list.<br /><br />自動翻訳: `true`の場合、ファブリストに全てのファブボタンが表示されます。 |
