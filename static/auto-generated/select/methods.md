### open

|                 |                                                                                                                                                                                                                                                                                                                                    |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| **Signature**   | `open(event?: UIEvent) => Promise<any>`                                                                                                                                                                                                                                                                                            |
