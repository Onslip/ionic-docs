### fixed

|                 |                                                                                                                                                         |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Description** | If `true`, the grid will have a fixed width based on the screen size.<br /><br />自動翻訳: `true`の場合、グリッドの幅は画面サイズに応じた固定幅となる。 |
| **Attribute**   | `fixed`                                                                                                                                                 |
| **Type**        | `boolean`                                                                                                                                               |
| **Default**     | `false`                                                                                                                                                 |
